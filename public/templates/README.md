# Export Templates

本目录存放浏览器端可读取的固定单据模板资产。

当前标准模板：

| 业务类型 | 标准模板 | 来源 |
|---|---|---|
| 出货单 | `sales-order-template.xlsx` | 由根目录 `出货单导出模版.xls` 转换 |
| 报价单 | `quote-template.xlsx` | 由根目录 `报价单导出模版.xls` 转换 |

使用规则：

- 业务代码只引用 `.xlsx` 标准模板。
- 原始 `.xls` 文件仅作为用户提供的原件备份，不作为运行时模板。
- 模板字段映射必须在 `Export Service` 内维护，页面层不得直接写单元格映射。
- 如用户后续替换模板，应保持本目录英文文件名稳定，避免影响代码引用。
