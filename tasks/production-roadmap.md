# 生产化推进路线图

本文档是 Tradgio 从本地 MVP 推进到生产可上线状态的主要执行依据。

## 1. 使用规则

- 全局任务从 `任务 26` 延续编号；第三轮专项优化继续使用 `OPT3-*`，两者互不占号。
- 所有任务默认状态为“待开始”；只有完成交付物、验证和文档同步后才能更新状态。
- 任务按 `P0 -> P1 -> P2 -> P3` 推进，前一阶段的准入门槛未通过时，不进入后一阶段。
- 任务 34 已通过 ADR-0002 锁定 IndexedDB 本地优先方案；静态托管供应商在任务 44 实测后确定。
- 当前仍是单人业务系统，不扩展多人协作、复杂权限、财务和库存预警。

状态枚举：`待开始`、`进行中`、`待验收`、`已完成`、`阻塞`。
