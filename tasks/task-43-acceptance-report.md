- 空数据库可升级到当前 schema，15 个 object store 与关键复合唯一索引创建成功；版本过新仍在写入前拒绝。
- 两次独立脱敏多账号迁移演练均通过，迁移后账号、业务实体、金额、库存、附件数量与字节摘要一致。

- 模板已缓存时可离线导出；模板或动态导出组件未缓存时显示明确联网提示。
- 错误密码、篡改、容量不足、事务失败和迁移失败不修改正式数据的既有专项测试全部回归通过。
